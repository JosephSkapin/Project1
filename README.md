# Project1
running the project goes as follows

LeakCount Program:
1. Run the make file just by executing "make" in the terminal inside of the Project1 Folder
2. then to execute the leak counter do "./leakcount ./testprogram"

sctracer program:
1. Make sure to run the make file in the terminal inside of the Project1 folder.
2. then to execute the program follow this format ./sctracer ./tracetest <outputfile_name> 
3. make sure the output file is a txt file so should have .txt at the end. EX: traceresult.txt
